2. Image Sprite

1. Special Naming convention will generate a new image sprite
2. Custom setting could be set in the _settings.sass
3. Naming convention: name.png, name_hover.png, name_active, name_target.png
4. Use the folderName + "-sprite", pass the name as the 1st arg to generate the sprite
5. Adding _active _target or _hover to the file name to add selectors for your sprites to generate selector like this: a:hover, a.glossy_hover, a.glossy-hover, a:target, a.glossy_target, a.glossy-target, a:active, a.glossy_active, a.glossy-active
